\thesubsection Data augmentation methods

\textcolor

green[change a name !!!] \textcolorgreen[two ways of using profile data] In contrastive learning for sentence embeddings, the sentence embedding model can better learn the representations from well-constructed paired examples. Such paired examples usually consists of the original example and the augmented example made by data augmentation techniques [zhang2022contrastive, wu2020clear] such as word deletion, reordering, substitution, and applying different dropout rate during embedding [gao2021simcse]. Such approaches are designed to enhance the modelâ€™s ability to learn representations and produce sentence embeddings that better represent the given textual content. The user-generated data will be accumulated along with the usage of the Edge LLM. Such data can be used to train the embedding model. Within such data, some data have explicit labels indicate the user preference regarding certain context. One the other hand, some data have implicit labels where such data in general contain the user-related information which can also help the Edge LLM to generate user-preferred content. In our work, we propose two data augmentation methods for the two different types of data.
